# Python Address Book

A simple and smart Python-based address book to store, manage, and organize your contacts efficiently.

---

## Features

- **Add Contact** – Save name, phone number (11 digits), Gmail, address, and mark as favorite.
- **Remove Contact** – Delete a contact by ID with confirmation.
- **Search Contact** – Search contacts by their unique ID.
- **View All Contacts** – Sorted list of contacts, separates favorites, normal, and deleted contacts.
- **Update Contact** – Modify name, number, Gmail, address, ID, or favorite status.
- **Clear Contacts** – Remove normal, favorite, or all contacts at once.
- **Automatic Saving** – All changes are saved in a `contacts.json` file.
- **Instruction Manual** – Easy guidance on how to use the program.
